# 快捷方式

> 管理快捷方式。
> 注意：您也可以使用 `Shortcuts` 应用。
> 更多信息：<https://support.apple.com/guide/shortcuts-mac/run-shortcuts-from-the-command-line-apd455c82f02/mac>。

- 运行指定的快捷方式（`统计假日`）：

`shortcuts run "{{统计假日}}"`

- 打印所有快捷方式：

`shortcuts list`

- 打印所有快捷方式文件夹：

`shortcuts list --folders`

- 在快捷方式编辑器中打开指定的快捷方式（`统计假日`）：

`shortcuts view "{{统计假日}}"`
