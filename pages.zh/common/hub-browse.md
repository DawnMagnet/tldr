# hub 浏览

> 在浏览器中打开 GitHub 仓库或打印 URL。
> 更多信息：<https://hub.github.com/hub-browse.1.html>。

- 在默认网页浏览器中打开当前仓库的主页：

`hub browse`

- 在默认网页浏览器中打开特定仓库的主页：

`hub browse {{owner}}/{{repository}}`

- 在默认网页浏览器中打开特定仓库的子页面，子页面可以是 "wiki"、"commits"、"issues" 或其他（默认： "tree"）：

`hub browse {{owner}}/{{repository}} {{subpage}}`
