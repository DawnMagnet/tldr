# pixi

> 项目的开发工作流程和环境管理。
> 更多信息：<https://pixi.sh>。

- 初始化一个新项目：

`pixi init {{path/to/project}}`

- 添加项目依赖：

`pixi add {{dependency1 dependency2 ...}}`

- 在项目环境中启动一个 pixi shell：

`pixi shell`

- 在项目环境中运行任务：

`pixi run {{task}}`

- 管理项目环境中的任务：

`pixi task {{command}}`

- 打印帮助信息：

`pixi {{command}} --help`

- 清理环境和任务缓存：

`pixi clean`
