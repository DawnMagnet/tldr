# bpython

> 一个华丽的 Python 解释器界面。
> 在 REPL 模式下提供语法高亮和许多其他不错的功能。
> 更多信息：<https://manned.org/bpython>。

- 启动 REPL（交互式 shell）：

`bpython`

- 执行特定的 Python 文件：

`bpython {{path/to/file.py}}`

- 执行特定的 Python 文件并启动 REPL：

`bpython --interactive {{path/to/file.py}}`

- 使用指定的 [c]onfig 文件，而不是默认配置：

`bpython --config {{path/to/file.conf}}`
