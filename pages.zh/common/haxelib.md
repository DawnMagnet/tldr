# haxelib

> Haxe 库管理器。
> 更多信息：<https://lib.haxe.org/>.

- 搜索 Haxe 库：

`haxelib search {{关键字}}`

- 安装 Haxe 库：

`haxelib install {{库名}}`

- 安装特定版本的 Haxe 库：

`haxelib install {{库名}} {{版本}}`

- 升级所有已安装的 Haxe 库：

`haxelib upgrade`

- 从 Git 仓库安装库的开发版本：

`haxelib git {{库名}} {{git_url}}`

- 卸载 Haxe 库：

`haxelib remove {{库名}}`

- 打印本地已安装的 Haxe 库树：

`haxelib list`
