# clifm

> 命令行文件管理器。
> 另见：`vifm`，`ranger`，`mc`，`nautilus`。
> 更多信息：<https://github.com/leo-arch/clifm>。

- 启动CliFM：

`clifm`

- 打开ELN（条目列表编号）为12的文件或目录：

`12`

- 创建一个新文件和一个新目录：

`n file dir/`

- 在当前目录中搜索PDF文件：

`/*.pdf`

- 选择当前目录中的所有PNG文件：

`s *.png`

- 删除之前选择的文件（使用`t`将文件发送到回收站）：

`r sel`

- 显示帮助：

`?`

- 退出CliFM：

`q`
