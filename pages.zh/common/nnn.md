# nnn

> 交互式终端文件管理器和磁盘使用分析器。
> 更多信息：<https://github.com/jarun/nnn>。

- 打开当前目录（或指定一个作为第一个参数）：

`nnn`

- 以详细模式启动：

`nnn -d`

- 显示隐藏文件：

`nnn -H`

- 打开已存在的书签（在`NNN_BMS`环境变量中定义）：

`nnn -b {{bookmark_name}}`

- 按照[ a ]表观磁盘使用情况 / [ d ]磁盘使用情况 / [ e ]扩展名 / [ r ]反向 / [ s ]大小 / [ t ]时间 / [ v ]版本排序文件：

`nnn -T {{a|d|e|r|s|t|v}}`

- 打开您选择的文件。选择文件后按`o`，然后输入打开文件的程序：

`nnn -o`
