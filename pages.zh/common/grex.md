# grex

> 生成正则表达式。
> 更多信息：<https://github.com/pemistahl/grex>。

- 生成一个简单的正则表达式：

`grex {{以空格分隔的字符串}}`

- 生成一个不区分大小写的正则表达式：

`grex -i {{以空格分隔的字符串}}`

- 用 '\d' 替换数字：

`grex -d {{以空格分隔的字符串}}`

- 用 '\w' 替换 Unicode 单词字符：

`grex -w {{以空格分隔的字符串}}`

- 用 '\s' 替换空格：

`grex -s {{以空格分隔的字符串}}`

- 为重复子字符串添加 {min, max} 量词表示：

`grex -r {{以空格分隔的字符串}}`
