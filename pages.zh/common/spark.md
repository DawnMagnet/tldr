# Spark

> Laravel Spark 命令行工具。
> 更多信息: <https://spark.laravel.com>。

- 注册您的 API 令牌：

`spark register {{token}}`

- 显示当前注册的 API 令牌：

`spark token`

- 创建一个新的 Spark 项目：

`spark new {{project_name}}`

- 创建一个带有 Braintree 桩的新 Spark 项目：

`spark new {{project_name}} --braintree`

- 创建一个带有团队计费桩的新 Spark 项目：

`spark new {{project_name}} --team-billing`
