# tr

> 转换字符：根据单个字符和字符集进行替换。
> 更多信息：<https://www.gnu.org/software/coreutils/tr>。

- 替换文件中所有出现的字符，并打印结果：

`tr {{查找字符}} {{替换字符}} < {{文件路径}}`

- 替换另一个命令输出中的所有字符：

`echo {{文本}} | tr {{查找字符}} {{替换字符}}`

- 将第一个集合中的每个字符映射到第二个集合中对应的字符：

`tr '{{abcd}}' '{{jkmn}}' < {{文件路径}}`

- 从输入中删除指定字符集的所有出现：

`tr -d '{{输入字符}}' < {{文件路径}}`

- 将一系列相同的字符压缩为一个字符：

`tr -s '{{输入字符}}' < {{文件路径}}`

- 将文件内容转换为大写：

`tr "[:lower:]" "[:upper:]" < {{文件路径}}`

- 从文件中剔除不可打印字符：

`tr -cd "[:print:]" < {{文件路径}}`
