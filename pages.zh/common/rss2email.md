# rss2email

> 将RSS源的新闻发送到电子邮件程序。
> 更多信息：<https://github.com/rss2email/rss2email>。

- 列出所有源：

`r2e list`

- 将RSS条目转换为电子邮件：

`r2e run`

- 添加一个源：

`r2e add {{feed_address}}`

- 使用特定的电子邮件地址添加源：

`r2e add {{feed_address}} {{new_email@example.com}}`

- 删除特定源：

`r2e delete {{number_of_feed_in_list}}`

- 显示帮助：

`r2e -h`
