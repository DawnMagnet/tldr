# gcloud 组件更新

> 将您安装的所有 Google Cloud CLI 组件更新到最新版本。
> 另见：`gcloud`。
> 更多信息：<https://cloud.google.com/sdk/gcloud/reference/components/update>。

- 将所有组件更新到最新版本：

`gcloud components update`

- 将所有组件更新到特定版本：

`gcloud components update --version={{1.2.3}}`

- 无需确认更新组件（适用于自动化脚本）：

`gcloud components update --quiet`
