# nth

> 命名哈希 - 立即识别任何哈希的类型。
> 更多信息：<https://github.com/hashpals/name-that-hash>。

- 命名哈希：

`nth -t {{5f4dcc3b5aa765d61d8327deb882cf99}}`

- 命名文件中的哈希：

`nth -f {{path/to/hashes}}`

- 以 JSON 格式打印：

`nth -t {{5f4dcc3b5aa765d61d8327deb882cf99}} -g`

- 在命名之前以 Base64 解码哈希：

`nth -t {{NWY0ZGNjM2I1YWE3NjVkNjFkODMyN2RlYjg4MmNmOTkK}} -b64`
