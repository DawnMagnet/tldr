# moe

> 一个用于ISO-8859-15编码文本的所见即所得文本编辑器。
> 更多信息：<https://www.gnu.org/software/moe/moe.html>。

- 打开moe并在保存编辑时创建备份文件（file~）:

`moe {{path/to/file}}`

- 以只读模式打开文件:

`moe --read-only {{path/to/file}}`

- 编辑文件而不创建备份:

`moe --no-backup {{path/to/file}}`

- 编辑文件时在搜索中忽略大小写:

`moe --ignore-case {{path/to/file}}`

- 保存并退出:

`<Ctrl> + X`
