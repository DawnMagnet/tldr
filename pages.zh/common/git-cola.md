# git cola

> 一款强大的 Git 图形用户界面，拥有流畅直观的用户界面。
> 更多信息请访问：<https://git-cola.readthedocs.io>。

- 启动图形用户界面：

`git cola`

- 以修改模式启动图形用户界面：

`git cola --amend`

- 提示输入 Git 仓库。默认为当前目录：

`git cola --prompt`

- 打开指定路径的 Git 仓库：

`git cola --repo {{path/to/git-repository}}`

- 将路径过滤器应用于状态小部件：

`git cola --status-filter {{filter}}`
