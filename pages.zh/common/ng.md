# ng

> 创建和管理 Angular 应用程序。
> 更多信息：<https://angular.io/cli>。

- 在一个目录中创建一个新的 Angular 应用程序：

`ng new {{project_name}}`

- 向应用程序添加一个新组件：

`ng generate component {{component_name}}`

- 向应用程序添加一个新类：

`ng generate class {{class_name}}`

- 向应用程序添加一个新指令：

`ng generate directive {{directive_name}}`

- 在应用程序的根目录中运行应用程序：

`ng serve`

- 构建应用程序：

`ng build`

- 运行单元测试：

`ng test`

- 显示当前 Angular 安装的版本：

`ng version`
