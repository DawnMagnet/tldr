# ptpython

> 一个更好的 Python REPL。
> 更多信息：<https://github.com/prompt-toolkit/ptpython>。

- 启动 REPL（交互式命令行）：

`ptpython`

- 执行指定的 Python 文件：

`ptpython {{path/to/file.py}}`

- 执行指定的 Python 文件并启动 REPL：

`ptpython -i {{path/to/file.py}}`

- 打开菜单：

`F2`

- 打开历史记录页面：

`F3`

- 切换粘贴模式：

`F6`

- 退出：

`<Ctrl> + D`
