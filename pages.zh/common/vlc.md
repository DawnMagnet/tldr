# vlc

> 跨平台多媒体播放器。
> 另见：`mpv`，`mplayer`，`ytfzf`。
> 更多信息：<https://wiki.videolan.org/Documentation:Command_line/>。

- 播放文件：

`vlc {{path/to/file}}`

- 全屏播放：

`vlc --fullscreen {{path/to/file}}`

- 静音播放：

`vlc --no-audio {{path/to/file}}`

- 循环播放：

`vlc --loop {{path/to/file}}`

- 从 URL 播放视频：

`vlc {{https://www.youtube.com/watch?v=oHg5SJYRHA0}}`
