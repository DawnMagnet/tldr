# Rails

> 一个用 Ruby 编写的服务器端 MVC 框架。
> 一些子命令如 `generate` 有其自己的使用文档。
> 更多信息：<https://guides.rubyonrails.org/command_line.html>。

- 创建一个新的 Rails 项目：

`rails new "{{project_name}}"`

- 在 3000 端口启动当前项目的本地服务器：

`rails server`

- 在指定端口启动当前项目的本地服务器：

`rails server -p "{{port}}"`

- 打开控制台以从命令行与应用程序交互：

`rails console`

- 检查当前 Rails 版本：

`rails --version`
