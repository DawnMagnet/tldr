# usql

> 通用的 SQL 数据库 CLI 接口。
> 更多信息：<https://github.com/xo/usql>。

- 连接到特定数据库：

`usql {{sqlserver|mysql|postgres|sqlite3|...}}://{{用户名}}:{{密码}}@{{主机}}:{{端口}}/{{数据库名称}}`

- 从文件中执行命令：

`usql --file={{路径/到/查询.sql}}`

- 执行特定的 SQL 命令：

`usql --command="{{sql_command}}"`

- 在 `usql` 提示符中运行 SQL 命令：

`{{提示符}}=> {{命令}}`

- 显示数据库模式：

`{{提示符}}=> \d`

- 将查询结果导出到特定文件：

`{{提示符}}=> \g {{路径/到/结果文件}}`

- 从 CSV 文件导入数据到特定表：

`{{提示符}}=> \copy {{路径/到/数据.csv}} {{表名}}`
