# uv python

> 管理 Python 版本和安装。
> 更多信息：<https://docs.astral.sh/uv/reference/cli/#uv-python>。

- 列出所有可用的 Python 安装：

`uv python list`

- 安装一个 Python 版本：

`uv python install {{version}}`

- 卸载一个 Python 版本：

`uv python uninstall {{version}}`

- 搜索一个 Python 安装：

`uv python find {{version}}`

- 将当前项目固定为使用特定的 Python 版本：

`uv python pin {{version}}`

- 显示 `uv` Python 安装目录：

`uv python dir`
