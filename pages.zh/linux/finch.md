# finch

> 基于控制台的模块化消息客户端。
> 更多信息：<https://developer.pidgin.im/wiki/Using%20Finch>。

- 启动 finch：

`finch`

- 退出：

`<Alt> + q 或 <Ctrl> + c`

- 显示操作菜单：

`<Alt> + a`

- 跳转到第 n 个窗口：

`<Alt> + {{number_key}}`

- 关闭当前窗口：

`<Alt> + c`

- 开始移动窗口，使用箭头键移动，完成后按 escape：

`<Alt> + m`

- 开始调整窗口大小，使用箭头键调整，完成后按 escape：

`<Alt> + r`
