# archlinux-java

> 在已安装的Java环境之间切换。
> 更多信息：<https://wiki.archlinux.org/title/Java#Switching_between_JVM>。

- 列出已安装的Java环境：

`archlinux-java status`

- 返回当前默认Java环境的短名称：

`archlinux-java get`

- 设置默认Java环境：

`archlinux-java set {{java_environment}}`

- 取消设置默认Java环境：

`archlinux-java unset`

- 修复无效/损坏的默认Java环境配置：

`archlinux-java fix`
