# nordvpn

> NordVPN 的命令行界面。
> 更多信息：<https://nordvpn.com/download/linux/>。

- 交互式登录到 NordVPN 账户：

`nordvpn login`

- 显示连接状态：

`nordvpn status`

- 连接到最近的 NordVPN 服务器：

`nordvpn connect`

- 列出所有可用国家：

`nordvpn countries`

- 连接到特定国家的 NordVPN 服务器：

`nordvpn connect {{德国}}`

- 连接到特定国家和城市的 NordVPN 服务器：

`nordvpn connect {{德国}} {{柏林}}`

- 设置自动连接选项：

`nordvpn set autoconnect on`
