# st

> 一个简单的 X Window 系统终端模拟器。
> 更多信息：<https://st.suckless.org>。

- 打开一个终端：

`st`

- 打开一个带有特定标题的终端：

`st -T {{title}}`

- 打开一个终端，执行给定命令，并将输出写入文件：

`st -o {{path/to/file}} -e {{command argument1 argument2}}`

- 增加/减少字体大小：

`<Ctrl> + <Shift> + {{Page Up|Page Down}}`

- 从剪贴板复制/粘贴：

`<Ctrl> + <Shift> + {{C|V}}`
