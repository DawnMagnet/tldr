# sar

> 监控各种Linux子系统的性能。
> 更多信息请访问：<https://manned.org/sar>。

- 报告发给物理设备的I/O和传输速率，每秒一次（按CTRL+C退出）：

`sar -b {{1}}`

- 报告共10个网络设备统计信息，每2秒一次：

`sar -n DEV {{2}} {{10}}`

- 报告CPU利用率，每2秒一次：

`sar -u ALL {{2}}`

- 报告共20个内存利用率统计信息，每秒一次：

`sar -r ALL {{1}} {{20}}`

- 报告运行队列长度和负载平均值，每秒一次：

`sar -q {{1}}`

- 报告分页统计信息，每5秒一次：

`sar -B {{5}}`
