# 嘟嘟声

> 一个用于发出PC扬声器声音的工具。
> 更多信息：<https://github.com/spkr-beep/beep>。

- 发出一个嘟嘟声：

`beep`

- 发出重复的嘟嘟声：

`beep -r {{重复次数}}`

- 以指定的频率（Hz）和持续时间（毫秒）发出嘟嘟声：

`beep -f {{频率}} -l {{持续时间}}`

- 将每个新的频率和持续时间作为单独的嘟嘟声播放：

`beep -f {{频率}} -l {{持续时间}} -n -f {{频率}} -l {{持续时间}}`

- 播放C大调音阶：

`beep -f {{262}} -n -f {{294}} -n -f {{330}} -n -f {{349}} -n -f {{392}} -n -f {{440}} -n -f {{494}} -n -f {{523}}`
