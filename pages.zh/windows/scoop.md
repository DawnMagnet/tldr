# scoop

> Scoop 包管理器。
> 一些子命令，如 `bucket`，有其自己的使用文档。
> 更多信息：<https://scoop.sh>。

- 安装一个包：

`scoop install {{package}}`

- 移除一个包：

`scoop uninstall {{package}}`

- 更新所有已安装的包：

`scoop update --all`

- 列出已安装的包：

`scoop list`

- 显示有关包的信息：

`scoop info {{package}}`

- 搜索一个包：

`scoop search {{package}}`

- 移除所有包的旧版本并清理下载缓存：

`scoop cleanup --cache --all`
